# Чат-бот для магазина лестниц

Чат-бот для консультирования клиентов и помощи в выборе лестниц и комплектующих.

## 🚀 Функционал
- Консультация по моделям лестниц (Престиж, Элегант, Престиж Мини и др.)
- Информация о материалах (хвоя, бук) и цветах RAL
- Помощь в подборе комплектации под параметры проема
- Расчет ориентировочной стоимости
- Отправка заявок в Telegram
- Сбор контактных данных (имя и телефон)
- AI-ответы на вопросы о продукции
